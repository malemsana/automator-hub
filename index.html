What is Automator Hub?

Automator Hub is a cross-platform desktop application built for developers, hobbyists, and power-users who need to automate tasks in the browser. It provides a simple UI to manage multiple, isolated browser profiles and a rich scripting environment to build powerful, data-driven automation agents.

  • Profile & Group Management

    Create isolated browser profiles. Organize them into colored groups to manage complex workflows with bulk actions.

  • Built-in Script Editor

    A powerful, built-in editor with syntax highlighting allows you to write and manage your Puppeteer-based scripts directly in the app.

  • Script-centric Database

    Every script gets its own private SQLite database, perfect for aggregating scraped data from multiple profiles into one location.

  • Secure Workspace (File I/O)

    Scripts can read and write files to a sandboxed workspace folder. Downloads are automatically saved here for easy access.

  • Backend Networking

    Empower your scripts with the ability to make external API calls. Your automations can now interact with any web service on the internet.

  • Custom Apps & Dashboards

    Scripts can create their own custom HTML dashboards, which appear in the "Apps" screen for data visualization and building custom control panels.

Scripting API Documentation

Every script runs in a Puppeteer environment and has access to the following global APIs.

The `automator` Object

`automator.getProfileName()`

Returns the name of the profile the script is currently running on.

const currentProfile = automator.getProfileName();
console.log(`Running on: ${currentProfile}`);

`automator.workspace` (File I/O)

Provides access to a secure, private folder for your script.

  • .getPath(): Returns the absolute path to the workspace folder.
  • .writeFile(filename, content): Saves data to a file in the workspace.
  • .readFile(filename): Reads data from a file in the workspace.

`automator.api` (Networking)

Allows scripts to make secure backend HTTP requests.

const response = await automator.api.fetch('https://api.example.com/data');
const jsonData = JSON.parse(response.body);

`automator.dashboard`

Allows a script to create its own custom UI, which will appear in the "Apps" screen.

const dashboardHtml = `<h1>My Report</h1>`;
await automator.dashboard.setHTML(dashboardHtml);

The `db` Object (Database)

Gives your script access to its own private SQLite database to store and aggregate data.

`db.execute(sql, [params])`

// Insert data safely
await db.execute('INSERT INTO results (item) VALUES (?)', ['Scraped data']);
const results = await db.execute('SELECT * FROM results');
